# Queue implementation by using LinkedList

Representation of Queue:

***Front --> ... --> Rear***

### Performance

```
Space Complexity - O(n) for n Nodes
```

```
Time Complexity
enQueue() - O(1)
deQueue() - O(1)
front() - O(1)
rear() - O(1)
isEmpty() - O(1)
contains() - O(n)
```
